1 INTRODUCCION

La visualización de datos constituye una pieza clave dentro del análisis estadístico y la toma de decisiones informadas. Mediante representaciones gráficas es posible reconocer patrones, tendencias y relaciones que de otro modo pasarían desapercibidas en un conjunto de datos.

En este documento se emplean diversos conjuntos de datos simulados para poner en práctica técnicas de visualización con la librería ggplot2 del lenguaje R. El propósito es mostrar distintas maneras de comunicar información a través de gráficos de dispersión, histogramas, diagramas de caja, líneas de tiempo y visualizaciones facetadas.

1.1 Dataset

Código 
library(ggplot2)

students <- data.frame(
  id = 1:20,
  hours_study = c(2,3,4,5,6,3,4,7,8,5,6,7,3,2,9,10,4,6,8,5),
  score = c(50,55,65,70,75,60,66,80,85,72,74,78,58,52,90,95,67,76,88,73),
  gender = rep(c("Male","Female"),10),
  major = rep(c("Engineering","Business","Arts","Science"),5)
)

2 ANALISIS DEL RENDIMIENTO DE ESTUDIANTES

2.1 DIAGRAMA DE DISPERSION HORAS DE ESTUDIO VS PUNTAJE

Crear un gráfico de dispersión que muestre la relación entre las horas de estudio y el puntaje obtenido.

Código 
ggplot(students, aes(hours_study, score)) +
  geom_point(color="blue", size=3) +
  theme_minimal() +
  labs(
    title="RELACION ENTRE HORAS DE ESTUDIO Y PUNTAJE",
    x="HORAS DE ESTUDIO",
    y="PUNTAJE"
  ) +
  theme(
    plot.title = element_text(hjust=0.5),
    panel.grid = element_line(color="#E6E6FA" )
  )

El gráfico evidencia una relación ascendente entre las horas dedicadas al estudio y los puntajes obtenidos. A medida que los estudiantes invierten más tiempo en preparar sus materias, sus resultados tienden a mejorar de forma consistente.

2.7 Relacion entre horas de estudio y puntaje segun la cerrera

Código 
ggplot(students, aes(hours_study, score)) +
  geom_point(size = 3, color = "#FFB6C1") +   
  geom_smooth(method = "lm", se = FALSE, color = "deeppink") +
  facet_wrap(~major) +
  theme_minimal(base_size = 15) +             
  labs(
    title = "Relación entre horas de estudio y puntaje según la carrera",
    x = "Horas de estudio",
    y = "Puntaje"
  ) +
  theme(plot.title = element_text(hjust = 0.5, size = 18))

El uso de facet_wrap() permite dividir el gráfico en paneles según la variable major, que representa la carrera de los estudiantes. Cada panel muestra de manera independiente la relación entre las horas de estudio y el puntaje dentro de una carrera específica. Esta organización facilita la comparación visual entre disciplinas, ya que permite identificar similitudes y diferencias en el comportamiento académico de los estudiantes según su área de estudio.

De esta manera, se facilita la comparación cruzada: es posible identificar patrones comunes o diferencias tanto entre géneros como entre áreas de estudio, ofreciendo una visión más completa y detallada del comportamiento académico.

2.9 Personalización estética

Código 
ggplot(students, aes(hours_study, score,
                     color = gender,
                     shape = major)) +
  geom_point(aes(size = score), alpha = 0.8) +
  geom_smooth(method = "lm", se = FALSE) +
  scale_color_manual(values = c("Male" = "blue", "Female" = "#FF69B4")) + # azul y rosado
  theme_minimal(base_size = 15) +
  labs(
    title = "Análisis multivariable del rendimiento académico",
    x = "Horas de estudio",
    y = "Calificación",
    color = "Género",
    shape = "Carrera",
    size = "Calificación"
  ) +
  theme(plot.title = element_text(hjust = 0.5, size = 18))

Este gráfico integra varias dimensiones de los datos en una sola visualización mediante atributos estéticos diferenciados. El color distingue el género, la forma de los puntos identifica la carrera y el tamaño refleja la calificación obtenida. Además, la línea de tendencia muestra cómo las horas de estudio se relacionan con el rendimiento académico. Esta combinación de elementos facilita un análisis multivariable, permitiendo observar patrones generales y, al mismo tiempo, comparar diferencias entre grupos de estudiantes según género y área de estudio.
